# Placement Portal Frontend

React + Vite frontend for the placement portal.

## Commands

```powershell
npm install
npm run dev
npm run lint
npm run build
```

The frontend expects the backend API at `http://localhost:8080/api` by default. Override it with:

```powershell
$env:VITE_API_URL="http://localhost:8080/api"
npm run dev
```

## Main Screens

- Login and register
- Admin dashboard
- Company dashboard
- Student dashboard
- Profile management
- Drive details
- Application history
